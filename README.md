# JARVIS
Jarvis is a voice assistant.

# REQUIRED MODULES
- datetime
- time
- speech_recognition
- webbrowser
- random
- pyttsx3
- os
- googletrans
- keyboard

# HOW CAN I USE .exe?
- Run "setup.bat" to access the application directly.

# REQUIRED PYTHON VERSION
- 3.9.7
